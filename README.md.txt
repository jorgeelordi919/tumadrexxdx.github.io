# Mi Primera Página Web

Este es un proyecto de prueba creado con HTML y CSS para aprender a publicar en internet usando GitHub Pages.
## 🛠 Tecnologías utilizadas
- HTML5
- CSS3

## 📖 Cómo usar este proyecto
1. Clonar el repositorio o descargarlo.
2. Abrir el archivo `index.html` en tu navegador.
3. (Opcional) Editar los archivos para personalizar la página.
